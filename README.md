# Australian Yacht Fleet Listing

A React application for browsing and booking luxury Australian yachts.

## Setup Instructions

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run the Application**
    ```bash
    npm run dev
    ```

3.  **Open in Browser**
    Navigate to `http://localhost:5173`.

## Features
- **Fleet Listing**: Browse available yachts.
- **Yacht Details**: View specs and request a booking.
- **Admin Dashboard**: View existing bookings at `/admin`.

## Technologies
- React + Vite
- React Router DOM
- CSS Modules / Vanilla CSS
